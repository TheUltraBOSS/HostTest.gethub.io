Sélecteurs simples

  1. Le sélecteur "nom d'élément"
    • Définition: Le sélecteur nom d'élément sélectionne les éléments HTML en fonction du nom de l'élément.
    • Exemple: Dans cette exemple ( cliquez ici) tous les éléments <p> de la page seront alignés au centre, avec une couleur de texte rouge.
  2. Le selecteur d'identifiant CSS "id"
    • L'identifiant d'un élément est unique dans une page, le sélecteur d'identifiant est donc utilisé pour sélectionner un élément unique!
    • Exemple: cliquez ici
    • Remarque: Un nom d'identifiant ne peut pas commencer par un nombre!
  3. Le selecteur d'identifiant CSS "class"
    • Le sélecteur de classe sélectionne les balises HTML avec un attribut de classe spécifique.
    • Pour sélectionner des balises avec une classe spécifique, écrivez le caractère point (.), Suivi du nom de la classe. Exemple: cliquez ici
    • On peut également spécifier que seuls des balises HTML spécifiques doivent être affectés par une classe. Exemple: cliquez ici. Dans cet exemple, seuls les éléments <p> avec class "center" seront rouges et alignés au centre.
    • Les balises HTML peuvent également faire référence à plusieurs classes. Exemple: cliquez ici. Dans cet exemple, la balise <p> sera stylisé selon class "center" et class "large".
    • Remarque: l'dentifiant d'une classe ne peut pas commencer par un nombre!
