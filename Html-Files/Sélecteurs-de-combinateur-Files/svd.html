Le selecteur adjacent


Cette forme de combinateur permet de sélectionner un élément uniquement si celui-ci
« suit » un élément donné et que les deux éléments sont les fils d'un même élément parent.

/* Ne cible que les paragraphes situé directement après une image */
            img + p {
              font-style: bold;
            }

Syntaxe

premier_element + element_cible { styles }

Exemple

CSS

li:first-of-type + li {
            color: red;
          }

HTML

<ul>
           <li>Un</li>
           <li>Deux</li>
           <li>Trois</li>
</ul>

Résultat

  • Un
  • Deux
  • Trois
