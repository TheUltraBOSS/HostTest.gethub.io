Sélecteurs de combinateur

  1. Le selecteur adjacent:
    • Définition: Le combinateur '+' permet de sélectionner les nœuds qui suivent immédiatement un élément donné.
    • Syntaxe : A + B.
    • Exemple: div + p permettra de cibler n'importe quel élément <p> qui suit immédiatement un élément <div>.

  2. Le sélecteur général adjacent
    • Définition: Le combinateur '~' permet de sélectionner les nœuds qui suivent un élément et qui ont le même parent.
    • Syntaxe : A ~ B.
    • Exemple: p ~ span permettra de cibler les éléments <span> qui suivent (immédiatement ou non) un élément <p> et qui ont le même élément parent.

  3. Le sélecteur d'élément enfant:
    • Définition: Le combinateur '>' permet de sélectionner les nœuds qui sont des enfants directs d'un élément donné.
    • Syntaxe : A > B
    • Exemple: ul > li permettra de cibler tous les éléments <li> qui sont directement situés sous un élément <ul>.

  4. Le sélecteurs d'élément descendant:
    • Définition: Le combinateur (espace) permet de sélectionner les nœuds qui sont des descendants (pas nécessairement des enfants directs) d'un élément donné.
    • Syntaxe : A B.
    • Exemple: div span permettra de cibler n'importe quel élément <span> situé à l'intérieur d'un élément <div>.

  5. Le combinateur de colonne:
    • Définition: Le combinateur || sélectionne les nœuds qui appartiennent à une colonne.
    • Syntaxe : A || B
    • Exemple: col || td permettra de cibler n'importe quel élément <td> sous la portée d'une colonne <col>.
