CHAPITRE3:Cascading Style Sheets

  1. INTRODUCTION:
    1. DEFENITION:
      • CSS décrit comment les éléments HTML doivent être affichés à l'écran, a l'impression ou sur d'autres media
      • CSS économise beaucoup de travail. Il peut contrôler la mise en page de plusieurs pages Web à la fois.
      • CSS est utilisé pour définir les styles de pages Web, y compris la conception, la mise en page et les variations d'affichage pour différents appareils et tailles d'écran.
    2. SYNTAXE:
      • la syntaxe CSS se compose d'un sélecteur et d'un bloc de déclaration
      • Le sélecteur pointe vers l'élément HTML que vous souhaitez styliser
      • Le bloc de déclaration contient une ou plusieurs déclarations séparées par des points-virgules.
      • Chaque déclaration comprend un nom de propriété CSS et une valeur, séparés par deux points.
  2. LES SELECTEURS CSS:
    1. Définition:

      2. Un sélecteur CSS sélectionne le ou les éléments HTML qu'on souhaite styliser.

    2. TYPES DE SELECTEURS
      • Sélecteurs simples: sélectionner les éléments en fonction du nom, de l'identifiant id ou de la classe
        • Le sélecteur "nom d'élément"
        • Le selecteur d'identifiant CSS "id"
        • Le selecteur d'identifiant CSS "class"
        • Le sélecteur CSS universel
        • Le sélecteur de regroupement CSS
      • Sélecteurs de combinateur : sélectionner les éléments en fonction d'une relation spécifique entre eux
        • Le sélecteur de descendant : Espace.
        • Le sélecteur enfant : >
        • Le selecteur adjacent : +
        • Le sélecteur général adjacent: ~
      • Sélecteurs de pseudo-classes : sélectionner des éléments en fonction d'un évenement
      • Sélecteurs de pseudo-éléments : sélectionner et styliser une partie d'un élément
      • Sélecteurs d'attributs: sélectionner des éléments en fonction d'un attribut ou d'une valeur d'attribut
